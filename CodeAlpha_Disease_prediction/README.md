Machine learning is transforming healthcare by enabling early disease detection and accurate diagnosis. This project develops a predictive system that analyzes structured medical data—such as age, symptoms, and clinical test results—to estimate disease likelihood.

Using datasets like Heart Disease, Diabetes, and Breast Cancer from the UCI Machine Learning Repository, various classification algorithms including SVM, Logistic Regression, Random Forest, and XGBoost are implemented and evaluated. The workflow involves data preprocessing, feature selection, model training, and performance assessment.

The best-performing model is deployed using Streamlit, allowing users to input medical data and receive real-time predictions through an interactive interface. This system demonstrates the effectiveness of machine learning in improving healthcare decision-making and enabling early diagnosis
