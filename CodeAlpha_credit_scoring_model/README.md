#  Credit Scoring Model

##  Overview
Predict an individual’s creditworthiness using past financial data.

##  Task
Build a classification model that evaluates credit risk and helps determine whether a loan application should be approved or rejected.

##  Approach
- Data preprocessing and handling missing values  
- Feature engineering from financial attributes  
- Model training using classification algorithms  
- Performance evaluation using standard metrics  

##  Algorithms Used
- Logistic Regression  
- Decision Tree  
- Random Forest  

##  Features Used
- Income  
- Loan Amount  
- Loan Term  
- Credit History  
- Other financial indicators  

##  Model Evaluation
The model is evaluated using:
- Accuracy  
- Precision  
- Recall  
- F1-Score  
- ROC-AUC  

##  Technologies Used
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- Streamlit  

##  Run the Project
pip install -r requirements.txt  
streamlit run app.py  

##  Conclusion
This project demonstrates how machine learning can be used to assess credit risk and support financial decision-making.
